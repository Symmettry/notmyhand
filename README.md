# Not My Hand!

Allows the player to manually select which poker hand to play when multiple valid hands are available, instead of the game choosing the highest priority one.

Adds a Change Hand button to the HUD that opens a menu listing all valid hands for the current selection. The chosen hand is then used for scoring.

Supports modded hands and integrates with the game’s existing hand evaluation system.

- Steamodded
